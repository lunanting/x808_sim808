/***command命令的结构说明***/
1.命令说明：
			1.total:Will show the total value;

			2.detail:Will show the detail value;

			3.limitup:You can input the sensor up-limit;

			4.restart:The system will be restart;

			5.back:Go back on one-up layer;

			6.help:Will show the command of the system;

			7.designer:Will show the designer of the system and his Individual Resume;

2.命令结构：
			我们将命令按照操作层面分类，这样可以直观的了解命令的使用（按照我们的目的）。
			第一层					第二层							第三层
			total									
			detail										
			limitup					"输入上限数据"					
			restart									
			help
			designer								
			back									
			back可以从高层返回到前面一层。help可以寻求当前命令层的帮助。
